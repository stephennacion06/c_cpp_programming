# External Resources

Curated list of external learning resources for C/C++ programming.

## üìö Books

### C Programming
- **The C Programming Language** by Brian Kernighan & Dennis Ritchie
  - *The* definitive C book by the language creators
- **C Programming: A Modern Approach** by K. N. King
  - Comprehensive and well-structured for learners
- **Expert C Programming: Deep C Secrets** by Peter van der Linden
  - Advanced topics and insights

### C++ Programming
- **The C++ Programming Language** by Bjarne Stroustrup
  - By the creator of C++
- **Effective C++** by Scott Meyers
  - 55 specific ways to improve your programs and designs
- **Effective Modern C++** by Scott Meyers
  - C++11 and C++14 best practices
- **C++ Primer** by Stanley Lippman
  - Comprehensive introduction
- **A Tour of C++** by Bjarne Stroustrup
  - Quick overview of modern C++

### Design and Best Practices
- **Clean Code** by Robert C. Martin
  - Writing maintainable code
- **Code Complete** by Steve McConnell
  - Comprehensive software construction guide
- **Design Patterns** by Gang of Four
  - Essential design patterns
